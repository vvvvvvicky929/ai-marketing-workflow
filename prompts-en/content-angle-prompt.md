# Content Angle Prompt

## Use Case

This prompt is used to generate marketing content angles for social media.

## Prompt

For a smart pet wearable product, generate 5 marketing content angles.

Please evaluate the product from these perspectives:

1. Health
2. Safety
3. Emotional connection
4. Lifestyle
5. Education

For each angle, include:

1. Angle name
2. User insight
3. Key message
4. Best platform
5. Example hook

Target audience:
Young U.S. pet owners.

## Expected Output

Please organize the answer in a table.

## My Review Criteria

I will review whether the content angles are platform-friendly, emotionally relevant, and suitable for TikTok and Instagram.
