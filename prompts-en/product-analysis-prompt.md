# Product Analysis Prompt

## Use Case

This prompt is used to analyze a smart product and identify its core marketing value.

## Prompt

You are an AI marketing strategist. Please analyze the following smart product and summarize:

1. Core product value
2. Key features
3. Target audience
4. User pain points
5. Purchase motivations
6. Possible marketing positioning

Product:
[Insert product description here]

## Expected Output

Please organize the answer in a clear table with concise but specific insights.

## Example Product

A smart pet wearable product that tracks pet location, activity, and health-related signals for young U.S. pet owners.

## My Review Criteria

I will review the AI output based on:

- Whether the product value is clear
- Whether the target audience is specific
- Whether the user pain points are realistic
- Whether the marketing positioning can support content creation
